# Project

### init(settings)
> Конструктор.

* **settings.staticDir**: Путь к директории статических файлов.
* **settings.dbURL**: Строка соединения с базой данных.

### addApplication(app)
> Добавляет приложение к проекту.

* **app**: Объект Application.

### start()
> Начинает прослушивание сервера.
