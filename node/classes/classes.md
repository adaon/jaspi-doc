classes
--------------------------------------------------------------------------------

Предоставляет функции для создания классов.

### createClass(props)
> Создает и возвращает функцию-конструктор класса.

* **props.init**: Функция инициализации экземпляра.
* **props.static**: Статические члены класса.

### Class(props)
> Синоним функции createClass.
