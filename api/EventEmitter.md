# EventEmitter

### on(name, handler)
> Добавляет обработчик события.

* **name**: Строковое имя события.
* **handler**: Функция-обработчик события.

### once(name, handler)
> Добавляет одноразовый обработчик события.

* **name**: Строковое имя события.
* **handler**: Функция-обработчик события.

### emit(name, args...)
> Генерирует событие.

* **name**: Строковое имя события.
* **args...**: Дополнительные аргументы, передаваемые обработчику события.
